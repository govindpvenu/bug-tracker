# Expo Bug Tracker

This project is a local-first Expo bug tracker. It keeps the original todo app behavior, but reframes the experience around reporting bugs, marking them open or closed, filtering the queue, and removing entries.

## Features

- Report a new bug from the quick capture input
- Toggle bugs between `Open` and `Closed`
- Filter the list by `All`, `Open`, or `Closed`
- Persist data locally with Expo SQLite-backed local storage
- Migrate existing todo entries into the new bug model automatically

## Get started

1. Install dependencies

   ```bash
   pnpm install
   ```

2. Start the app

   ```bash
   pnpm start
   ```
